checkMode()¶

Test to see if the supply is in constant current mode or constant voltage mode returns a string ‘CV’ if constant voltage and ‘CC’ if the mode is constant current.

closePort()¶
current(setCurrent='None')¶

If no arguments are passed this function queries the current and returns it. otherwise it will try to set the current of the powersupply and return a None type. This is very similar to the voltage() function but has slightly differet formatting.

The current is set in AMPS with four digits after the decimal (minimum precision = 0.1 mA).

The returned current float is also in AMPS.
